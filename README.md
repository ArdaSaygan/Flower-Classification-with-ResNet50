# Flower-Classification-with-ResNet50

Worked with ["102 Category Flower Dataset"](https://www.robots.ox.ac.uk/~vgg/data/flowers/102/) 

Used Transfer-Learning with ResNet50 pretrained model


train accuracy :  0.91

val val_accuracy :  0.87
